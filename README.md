# 🧝 Diana - Interactive Assistant for ComfyUI

<img src="https://github.com/Rimor-dev/my-assets_/blob/main/assets/dem2.gif" alt="Diana" width="720">

Diana is an interactive assistant that lives in your ComfyUI workspace. She reacts to queue events, speaks phrases, and switches personalities through different libraries.

## ✨ Features

- 🎭 **Multiple Personalities**: Diana, Kitty, Kerrigan (more coming!)
- 🔊 **Voice Reactions**: Audio feedback for queue events (start, complete, error)
- 💭 **Text Bubbles**: Random phrases for each personality
- 🎨 **Animated Sprites**: Smooth animations for different emotions
- 📌 **Draggable & Pinable**: Move her anywhere, pin to corner
- 🔇 **Audio Toggle**: Click the speaker to mute/unmute
- ⏱️ **Idle Timer**: Random reactions after 11-13 minutes of inactivity
- 🎯 **Smart Error Detection**: Detects both explicit errors and failed generations

# 📦 Installation

```bash
cd ComfyUI/custom_nodes/
git clone https://github.com/Rimor-dev/comfyui-diana.git
```

# 📥 Adding New Libraries
Place .lib files in the libraries/ folder. Each library contains:
Animated sprites (128x128 PNG frames)
Audio clips (MP3 format)
Personality phrases
Color themes

# 🎯 How to Use
Start ComfyUI - Diana appears automatically
First click anywhere activates her
Run a workflow - she reacts to:
🔥 Start of generation
✅ Successful completion
❌ Error or failed generation
Click speaker to toggle audio on/off
Drag header to move her around
Click 📌 to pin to corner
Switch libraries with ◀ ▶ buttons
